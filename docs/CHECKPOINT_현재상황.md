## 🛠️ 실제 API 연동 준비 상태

### **준비 완료 항목:**
```
✅ 엔진: tojeong-engine.js
✅ 프롬프트: tojeong-prompt.js (가정운, 여행운 포함)
✅ 프론트엔드: tojeong-test.html (가정운, 여행운 버튼 포함)
✅ Mock API: 6개 카테고리 데이터
✅ 데이터: tojeong-gua-144.json
```

### **연동 방법 (Phase 2):**
```javascript
// server.js에 추가
const tojeongEngine = require('./engines/core/tojeong-engine');
const { getTojeongPrompt } = require('./backend/prompts/tojeong-prompt');
const claudeAPI = require('./backend/claude-api');

app.post('/api/tojeong', async (req, res) => {
  const { year, month, day, isLunar, targetYear, category } = req.body;
  
  // 1. 엔진으로 144괘 계산
  const engineResult = tojeongEngine.calculate({
    year, month, day, isLunar
  }, targetYear);
  
  // 2. 프롬프트 생성 (6개 카테고리 모두 지원)
  const prompt = getTojeongPrompt(category, engineResult);
  
  // 3. Claude API 호출
  const aiResult = await claudeAPI.call(prompt);
  
  // 4. 결과 반환
  res.json({
    success: true,
    year: targetYear,
    category: category,
    mainGua: engineResult.mainGua,
    ...aiResult
  });
});
```
