## 3. 전체 프롬프트 공통 분석

### 3.1 공통 강점

**1. 일관된 구조**
- 모든 프롬프트가 동일한 패턴
- 역할 정의 → 데이터 → 해석 지침 → 작성 스타일

**2. 명확한 분량 제어**
- 대부분 300~500자 (일부는 100~150자)
- 타겟 사용자 고려 (55-65세 여성)

**3. 구체적인 해석 지침**
- 최소 4가지, 평균 6가지
- 단계별로 무엇을 분석할지 명확

**4. 동적 데이터 활용**
- daeun: 대운 리스트, 현재 나이
- sinsal: 신살 객체
- taekil: 택일 결과, 목적
- move/travel: 역마살 유무
- love: 성별

**5. 작성 스타일 명시**
- "구체적으로", "현실적으로", "긍정적으로"
- 타겟 사용자 고려

---

### 3.2 공통 약점

**1. Few-shot 예시 부족**
- 모든 프롬프트에 예시 답변 없음
